# Virtual Piano Simulator

A simple virtual piano simulator built with HTML, CSS, and JavaScript.

## Features

- Interactive piano keys
- Keyboard support
- Volume control
- Toggle key labels

## Technologies Used

- HTML
- CSS
- JavaScript

## Setup

1. Clone the repository:

    ```bash
    git clone [repository URL]
    ```

2. Open `index.html` in your browser.
